前端学习的4个网站

大部分的人心中都有一个编程梦，能使用自己编写的软件，那将是一件多么cool的事。真正选择从事程序员行业的只是诸多梦想家中的践行者。梦想家们以诸如：“我数学不好”、“我逻辑太差”、“我英语不行”的借口使自己宽慰。实际上在真正开发过程中，可能更需要的其他的一些品质。比如：坚持、求新、思考、发量。

前端是一门包括html/css、JavaScript的语言，入门比较快，能在很短的学习周期内建立第一个静态页面，在继续学习的过程中，你甚至更创建一个无比炫酷的网站。

这次推荐4个能够系统学习前端的网站。

1.W3shool

链接：https://www.w3school.com.cn/

W3shool的前端基础知识比较全面，非常适合作为新手的入门进程。它提供在线实例测试工具。根据w3cshool规划的目录，可以很快的对前端有一个初步的认识。

令人遗憾的是W3shool在解释某些内容时存在一些错误，而且内容比较少，不能深入学习。

2.Techbrood

链接：https://techbrood.com/h5b2a?p=index

Techbrood适用于对前端有了初步认识的朋友进阶，适合精细化整理前端知识。

该网站的内容包含了：

  • web简介
  • html5入门
  • css3入门
  • svg入门
  • JavaScript入门
  • css3进阶
  • html5进阶
  • ……
  • Techbrood的

3.IT 修真院

链接：http://www.jnshu.com/occupation/1/task

IT修真院属于培训机构。我并非是推荐大家去报班学习，我们可以利用修真院的任务来锻炼我们已经学习过的前端技术，而且可以通过提交任务让师兄免费为我们检查代码并给出他们的意见。

对于初学者而言，修真院前期的任务比较难，可以通过其他人的提问及回答来丰富自生的知识。

在学习中修真院并不提倡看视频学习而是要去在实际操作中查阅资料，这个观点要根据我们自己的实际情况来作出反应。

ps.如果大家觉得必须要抱一个培训班，建议多横向对比，多去知乎、简书等平台看看已经培训过建议。

4.MDN web docs

链接：https://developer.mozilla.org/zh-CN/

没什么说的，MDN是前端最全面、最系统、更新最快的文档学习地方。
